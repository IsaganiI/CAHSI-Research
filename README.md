# Predicting RNA and CTCF Protein Interaction

## Problem Description
The project aims to predict which RNAs interact with the protein CTCF. There are previous models such as deep learning and traditional ML models to predict RNA-CTCF interaction. These models achieved about 81% accuracy. We will explore whether graph neural networks can deliver better performance.

## Research Significance
CTCF is a protein involved in 3D genome organization. 3D genome organization is essential for proper genome function and gene regulation. CTCF localization depends on its interaction with RNAs. Predicting which RNAs that interact with CTCF can be very helpful.

## Research Goals and Objectives
- **Aim 1:** Validate previous models that predict CTCF-RNA interaction.
- **Aim 2:** Design new model based on Graph Neural Networks to predict RNA-CTCF interaction.
